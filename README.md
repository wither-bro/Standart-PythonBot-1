## 🛠 Технологии внутри:
*   **Язык:** Python 3.10+
*   **Движок:** `aiogram 3.x` (Самый мощный и современный)
*   **Архитектура:** Разделение на файлы (Config, Handlers, Keyboards)
*   **Автоматизация:** GitHub Actions (.yml)
